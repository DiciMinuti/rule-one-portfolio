# Rule #1 Portfolio App Docs

Internal planning docs for the free, unauthenticated Rule #1 research app.

## Files

- `00-product-brief.md` - product intent, constraints, scope, and principles.
- `01-rule1-method.md` - Rule #1 method, app lessons, formulas, and caveats.
- `02-ui-ux-spec.md` - complete UI and UX plan for the app.
- `03-data-and-calculation-plan.md` - free data plan, normalized data model, calculation pipeline.
- `04-build-roadmap.md` - implementation sequence for the Next.js app.

## North Star

Build a private, free-first research cockpit for evaluating businesses through the Rule #1 lens:

1. Understand the business.
2. Check Meaning, Moat, Management, and Margin of Safety.
3. Calculate the Big Five and valuation transparently.
4. Save good businesses to a local saves list.
5. Teach the method in context, without turning the app into a course website.
